기본 속성 선택자

        - 요소에 작성된 속성을 이용헤 선택하는 선택자

        - 속성, 속성 값이 일치하는 요소만을 선택

        [작성법]
         
            1) 같은 속성을 가지는 요소 선택
                [속성명] { css 코드; }
            2) 같은 속성, 속성 값을 가지는 요소 선택
                [속성명=속성값] { css 코드; }

                * 속성 값은 '',""로 또는 안 감싸져도 문제 없음

반응 선택자

        - 사용자의 움직임에 반응하여 요소를 선택하는 선택자

        선택자:hover : 선택된 요소에 마우스가 올라갔을 때 + 마우스기 벗어나면 원래 상태로 복원 됨

        선택자:active : 선택된 요소를 클릭하고 있을 경우 선택
    
        - input 태그의 상태에 따라서 선택되는 선택자

        :focus
            - input 요소에 초점이 맞춰졌을 때 선택
             (즉, 입력할 수 있는 상태가 되었을 때)

        :checked
            - type이 radio, checkebox인 요소가 체크되었을 때 선택

        :enabled / :disabled
            - input에 값을 작성할 수 있을 때/없을 때

동위 선택자(+,~)

        동위(동등한 위치 == 형제관계) 관계에서 다음(뒤, 아래)에 위치한 요소를 선택하는 선택자

        1) A 바로 다음에 위치한 요소를 하나 선택(+)
            
                A선택자 + B선택자 { CSS 코드; }
            
        
            2) A 다음에 위치한 모든 B요소를 선택(~, Tild(틸드))
            
                A선택자 ~ B선택자 { CSS 코드; }
