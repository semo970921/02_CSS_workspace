CSS (Cascading Style Sheets)

        Cascading: 종속, 여러 단으로 된 폭포

        [CSS]
        * 마크업 언어(HTML)가 실제 화면에 표시되는 방법을 기술한 언어

        * 하나의 요소(태그 처음부터 끝)에 여러 CSS 코드를 겹쳐서 원하는 스타일을 표시
    
        [용어 정리]
        id(식별자) 속성
            - 하나의 HTML 파일 내에 딱 1개만 존재할 수 있음(중복X)
                -> 요소를 구분하는 속성

            - id를 나타내는 기호 : #

        class(학급, 모입) 속성
            - 요소 하나가 여러 모임(class)에 참여 가능
                -> 같은 class를 갖는 요소가 존재할 수 있음

            - class를 나타내는 기호: .

CSS 선택자(Selector)

        - html 파일 내에서 스타일을 적용하고 싶은 요소를 선택하는 방법

태그 선택자

        - 같은 태그를 모두 찾아서 선택

            [작성법]
            태그명 { CSS 코드; }

ID(#) 선택자

        - HTML 문서 내에 같은 id를 가지는 요소를 선택

            [작성법]
            #id속성값 { css 코드;}

clss(.) 선택자

        - HTML 문서 내에서 같은 class 속성 값을 지니는 요소를 모두 선택

            [작성법]
            .class속성값 { css 코드; }
